Questions Fréquemment Posées

Qu'est-ce que Teapot ?

Teapot est une application qui vous permet de stocker dans une liste personnalisée les thés que vous possédez chez vous. Vous pouvez les entrer manuellement ou les trouver dans notre base de données pré-remplie.

Comment puis-je accéder à l'application ?

Pour accéder à l'application, vous devez vous connecter en enregistrant un pseudo, votre adresse e-mail et un mot de passe.

Comment puis-je voir la liste de mes thés ?

Vous pouvez voir la liste de tous vos thés de manière efficace grâce à l'onglet "Liste" de l'application.

Comment puis-je voir les détails de chaque thé ?

En cliquant sur chaque thé dans la liste, vous aurez la possibilité de le sélectionner. Ensuite, en allant sur l'onglet "Timer", vous retrouverez les thés que vous avez sélectionnés au préalable pour lancer le minuteur. Le minuteur démarrera pour tous les thés sélectionnés en même temps.
