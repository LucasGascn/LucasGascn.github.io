FAQ - Teapot

Découvrez comment utiliser notre application mobile pour gérer votre collection de thés.

Questions Fréquemment Posées

Qu'est-ce que Teapot ?

Teapot est une application qui vous permet de stocker dans une liste personnalisée les thés que vous possédez chez vous. Vous pouvez les entrer manuellement ou les trouver dans notre base de données pré-remplie.

Comment puis-je accéder à l'application ?

Pour accéder à l'application, vous devez vous connecter en enregistrant un pseudo, votre adresse e-mail et un mot de passe.

Par la suite vous devrez vous connecter en utilisant comme identifiant soit votre pseudo soit le mail que vous avez enregistré lors de votre inscription

Comment puis-je voir la liste de mes thés ?

Vous pouvez voir la liste de tous vos thés de manière efficace grâce à l'onglet "Liste" de l'application.

Comment puis-je voir les détails de chaque thé ?

En cliquant sur chaque thé dans la liste, vous aurez un visuel simple des détails de votre thés mais vous aurez aussi un bouton détail vous permettant d'accédez à la totalité des information du thé sélectionné

Comment puis-je lancer un minuteur?

En cliquant sur chaque thé dans la liste, vous aurez la possibilité de le sélectionner. Ensuite, en allant sur l'onglet "Timer", vous retrouverez les thés que vous avez sélectionnés au préalable pour lancer le minuteur. Le minuteur démarrera pour tous les thés sélectionnés en même temps.

Comment puis-je ajouter un thé à ma liste

Pour ajouter un thé à votre liste personnel vous avez deux possibilitées. la première est de le rentré manuellement, vous pourrez choisir alors le nom de votre thé, son type, la manière de stockage, le moment de la journée ou il est préférable de le consommer, si il est bio ou non, sa température, son temps d'infusion, les différents ingrédients qu'il possède, vous serez aider par notre base de donnée qui possède déjà un grand nombre d'ingrédient différent, et enfin le pays d'où provient votre thé.

La deuxième manière de faire est beaucou plus imple, si vous posséder un thé d'une de dnos marque partenaire vous aurez de grande chance de retrouver ce thé dans notre base de donnée et donc de l'ajouter dans votre liste en quelque clique seulement.
